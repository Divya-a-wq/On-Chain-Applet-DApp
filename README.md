# WeilChain AppletHub  
## A Decentralized Applet Marketplace on WeilChain

Discover, execute, and compose on-chain applets with wallet-based authentication, transparent execution history, and a monetizable ecosystem for creators.

---

## ğŸ“– Overview

WeilChain AppletHub is a decentralized application (dApp) marketplace built on **WeilChain** that enables users to discover, execute, and compose small reusable on-chain programs called *applets*. The platform provides secure wallet-based authentication, transparent on-chain execution history, and a fair monetization model for applet creators.

The goal is to build an **on-chain App Store** that promotes decentralization, composability, and trustless execution.

Installation and Setup

Step 1: Clone the repository

git clone https://github.com/your-username/weilchain-applet-dapp-ui.git

cd weilchain-applet-dapp-ui

Step 2: Install dependencies

npm install

Step 3: Run the application

npm run dev

Open the application in browser:

http://localhost:5173

Application Flow

User connects wallet

Applets are displayed from the marketplace

User selects an applet

Wallet signs the transaction

Applet executes on-chain

Execution history is recorded on blockchain

Creator receives payment automatically
